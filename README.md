# GreenPulse: Climate Change Data Insights

## Project Overview

GreenPulse is a Data Analytics project focused on analyzing global climate change indicators between 2000 and 2024.

## Objectives

* Analyze climate trends
* Study CO₂ emissions and temperature changes
* Perform data cleaning and preprocessing
* Create meaningful visualizations
* Apply Machine Learning using Linear Regression

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Dataset Features

* Year
* Country
* Average Temperature
* CO₂ Emissions
* Sea Level Rise
* Rainfall
* Population
* Renewable Energy (%)
* Extreme Weather Events
* Forest Area (%)

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Exploration
4. Data Analysis
5. Data Visualization
6. Machine Learning
7. Conclusion

## Key Outcomes

* Identified climate trends across countries
* Analyzed relationships between CO₂ emissions and temperature
* Generated visual insights using charts and heatmaps
* Built a Linear Regression model for predictive analysis
